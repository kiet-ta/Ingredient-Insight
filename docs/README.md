## Muc tieu tai lieu

1. Giai thich toan bo luong chay cua code tu luc game load mod den luc nguoi choi hover item.
2. Trinh bay mindset lam DST mod on dinh, tranh crash va tranh input leak.
3. Chuan hoa quy trinh debug, test, release theo tieu chuan ky thuat.
4. Ho tro onboarding nhanh cho contributor moi.
